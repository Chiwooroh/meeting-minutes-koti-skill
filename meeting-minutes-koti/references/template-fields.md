# 회의록 양식 필드

번들 양식 `assets/회의록_양식.hwp`의 핵심 필드는 다음과 같다.

| 필드 | 작성 방식 |
| --- | --- |
| 과제명 | 사용자가 제공한 과제명, 폴더명, 기존 문서에서 확인한 과제명을 우선 사용한다. 불명확하면 `[확인 필요: 과제명]`으로 둔다. |
| 회의명 | 녹음 폴더명 또는 회의 주제를 바탕으로 `○○ 회의` 형식으로 작성한다. |
| 개최일시 | 녹취 발언, 파일/폴더명, 사용자 제공 정보에서 확인한다. 추정일은 표시하지 말고 확인 필요로 둔다. |
| 장소 | transcript나 사용자 제공 정보에 있을 때만 쓴다. 화상회의면 `온라인 회의` 등으로 명시한다. |
| 참석인원 | 참석자 이름/소속을 가능한 한 나열한다. 수만 알면 `총 N명`으로 쓴다. |
| 회의목적 | 한 문장으로 작성한다. `...을 논의하고 향후 추진방향을 정하기 위함` 같은 공문서식 표현을 사용한다. |
| 회의내용 | 안건별 요약으로 작성한다. 대화 순서보다 보고서 가독성을 우선한다. |

## 출력 Markdown 골격

```markdown
# 회의록

| 구분 | 내용 |
| --- | --- |
| 과제명 |  |
| 회의명 |  |
| 개최일시 |  |
| 장소 |  |
| 참석인원 |  |
| 회의목적 |  |

## 회의내용

□ 안건 1
- 내용
  - 세부 내용

□ 안건 2
- 내용
  - 세부 내용

## 확인 필요 사항

- 없음
```
