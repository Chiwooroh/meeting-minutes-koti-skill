# 공무원 보고형 회의록 작성 기준

## 문체

- 문장은 간결하게 작성하고, 회의록 본문은 `~함`, `~필요`, `~예정`, `~검토` 중심으로 정리한다.
- 발화자별 대화록이 아니라 안건별 논의 결과 중심으로 쓴다.
- "좋습니다", "네", "음", 반복 설명, 잡담, 장비 점검 발언은 제외한다.
- 단정할 수 없는 내용은 추측하지 말고 `[확인 필요: 내용]`으로 표시한다.

## 내용 선별

반드시 반영할 내용:

- 회의 목적, 주요 안건, 쟁점
- 합의 사항, 결정 사항, 보완 요청
- 담당 기관/담당자, 후속 조치, 일정
- 법령, 지침, 연구과제명, 산출물명, 예산/계약/일정 관련 언급

축약하거나 제외할 내용:

- 같은 내용의 반복 발언
- 의례적 인사, 회의 시작/종료 멘트
- 안건과 무관한 사담
- 발화자의 말투나 감탄사

## 본문 구성

`회의내용`은 다음 구조를 기본으로 한다.

```markdown
□ 주요 안건명
- 핵심 논의 결과
  - 세부 근거 또는 보완 의견
  - 필요한 후속 조치

□ 추가 검토 사항
- 검토 필요 내용
  - 담당/기한이 있으면 명시
```

## 품질 점검

- 회의 목적과 회의내용이 서로 대응되는지 확인한다.
- 참석자, 일시, 장소가 없으면 임의 작성하지 않는다.
- transcript에 없는 결론을 만들어내지 않는다.
- 최종 문서에는 녹취록 원문을 길게 붙이지 않는다.
